## 购买注意事项

### 1. 必须选择固定公网IP
- 微信公众号API需要IP白名单
- **按流量计费** 通常有固定公网IP
- **按带宽计费** 也提供公网IP
- 购买时确认："公网IP" 或 "弹性公网IP"

### 2. 操作系统选择
- **推荐：Ubuntu 22.04 LTS**（最稳定，社区支持好）
- 备选：Ubuntu 20.04 LTS、CentOS 7+
- **不推荐：Windows Server**（贵、占用资源多、没必要）

### 3. 地域选择
选择离你最近的地域，降低网络延迟：
- 华南：广州、深圳
- 华东：上海、杭州
- 华北：北京、天津

### 4. 带宽选择
- **按带宽计费**：适合流量稳定的场景（如3Mbps ≈ ¥20/月）
- **按流量计费**：适合流量波动大的场景（如1GB ≈ ¥0.8）
- **推荐：按流量计费 + 峰值带宽3-5Mbps**

### 5. 优惠活动
- **新用户首单优惠**：通常有5折甚至更低
- **学生优惠**：阿里云"云翼计划"、腾讯云"校园优惠"
- **长期套餐**：买1年可能有折扣

## 购买后需要做的事

### 1. 获取服务器信息
购买完成后，你会得到：
- **公网IP地址**：如 `123.45.67.89`
- **登录用户名**：通常是 `root` 或 `ubuntu`
- **登录密码**：购买时设置或后续重置

### 2. 测试连接
```bash
# 打开终端（Windows用PowerShell或cmd）
ssh root@你的服务器IP
# 例如：ssh root@123.45.67.89
```

首次连接会提示：
```
The authenticity of host '123.45.67.89' can't be established.
...
Are you sure you want to continue connecting (yes/no)?
```
输入 `yes` 并回车

然后输入密码（输入时不会显示，这是正常的）

### 3. 重置密码（如果需要）
- 腾讯云：控制台 → 实例列表 → 更多 → 密码/密钥 → 重置密码
- 阿里云：ECS控制台 → 实例 → 更多 → 密码/密钥 → 重置实例密码
