## 📋 接下来你需要做的（按顺序）

### 第1步：购买云服务器
**推荐：** 腾讯云轻量应用服务器
- CPU：1核
- 内存：2GB
- 硬盘：40GB SSD
- 带宽：3-5Mbps（按流量计费）
- 系统：Ubuntu 22.04 LTS
- 预算：¥50-80/月

**购买链接：**
- 腾讯云：https://cloud.tencent.com/product/lighthouse
- 阿里云：https://www.aliyun.com/product/ecs

**完成后获得：**
- ✅ 服务器公网IP（如：`123.45.67.89`）
- ✅ 登录用户名（通常是 `root` 或 `ubuntu`）
- ✅ 登录密码

---

### 第2步：上传文件到服务器

#### 方法A：使用 SCP（推荐，最简单）
在**本地电脑**打开 PowerShell 或终端，执行：
```bash
# 进入项目目录
cd c:\Users\Administrator\WorkBuddy\20260427080318\wechat_auto_publish

# 上传文件到服务器
scp main.py config.py requirements.txt root@你的服务器IP:/opt/wechat_auto_publish/
```

#### 方法B：使用 Git
```bash
# 在 GitHub/Gitee 创建私有仓库
git init
git add .
git commit -m "Initial commit"
git remote add origin your_repo_url
git push -u origin main

# 在服务器上克隆
ssh root@你的服务器IP
cd /opt/wechat_auto_publish
git clone your_repo_url .
```

#### 方法C：手动创建
SSH 登录服务器后，使用 `nano` 或 `vim` 手动创建文件并粘贴内容。

---

### 第3步：运行部署脚本

SSH 登录服务器后执行：
```bash
# 上传 deploy.sh 到服务器
# 然后执行：
chmod +x deploy.sh
./deploy.sh
```

**部署脚本会自动完成：**
1. ✅ 检测操作系统
2. ✅ 获取服务器公网IP（**需添加到微信白名单**）
3. ✅ 安装 Python3、pip、git
4. ✅ 创建项目目录 `/opt/wechat_auto_publish`
5. ✅ 创建 Python 虚拟环境
6. ✅ 安装依赖（requests、pillow）
7. ✅ 检查配置文件
8. ✅ 测试运行
9. ✅ 配置定时任务（Crontab）
10. ✅ 配置日志轮转

---

### 第4步：添加IP白名单（关键！）

**此步骤必须完成，否则微信API调用会失败！**

1. 登录微信公众平台：https://mp.weixin.qq.com
2. 进入 **设置与开发** → **基本配置**
3. 找到 **IP白名单**，点击 **修改**
4. 点击 **添加**，输入你的**服务器公网IP**
   - 部署脚本会显示你的服务器IP
   - 也可以登录服务器执行：`curl ifconfig.me`
5. 保存并等待生效（约1-5分钟）

---

### 第5步：验证部署

#### 手动测试运行
```bash
ssh root@你的服务器IP
cd /opt/wechat_auto_publish
source venv/bin/activate
python main.py
```

**检查点：**
- ✅ 脚本正常运行，无报错
- ✅ 登录微信公众平台，查看 **草稿箱** 是否有新草稿

#### 查看定时任务
```bash
crontab -l
# 应该看到类似这样的输出：
# 0 8 * * * cd /opt/wechat_auto_publish && source venv/bin/activate && python main.py >> /var/log/wechat_publish/publish.log 2>&1 # wechat_auto_publish
```

#### 查看日志
```bash
tail -f /var/log/wechat_publish/publish.log
```

## 🔧 故障排查

### 问题1：定时任务未执行
```bash
# 检查 cron 服务是否运行
sudo systemctl status cron    # Ubuntu
sudo systemctl status crond   # CentOS

# 启动 cron 服务
sudo systemctl start cron
sudo systemctl enable cron
```

### 问题2：微信API报错 `invalid ip`
**原因：** 服务器IP未添加到白名单
**解决：**
1. 登录服务器，获取公网IP：`curl ifconfig.me`
2. 登录微信公众平台，添加IP到白名单

### 问题3：Python依赖缺失
```bash
cd /opt/wechat_auto_publish
source venv/bin/activate
pip install -r requirements.txt
```

### 问题4：配置文件错误
```bash
# 检查 config.py 是否填写正确
nano /opt/wechat_auto_publish/config.py

# 确保没有 "YOUR_APPID_HERE" 这样的占位符
```
